## workflows الحالية

- `Start Expo`: يشغل Expo web على port 5000.
- `Start application`: يشغل لوحة الإدارة Next.js على port 3000.

## خريطة تعديل سريعة

- تعديل تبويبات أو أدوار التطبيق: `app/(tabs)/_layout.tsx`.
- تعديل المصادقة أو fallback: `contexts/AuthContext.tsx`, `services/supabaseClient.ts`.
- تعديل منطق إنشاء الطلب: `contexts/AppContext.tsx`, `services/ordersService.ts`, `services/chatEngine.ts`.
- تعديل خدمة شات محددة: `services/chatEngineModules/<service>.ts` و`services/chatFlows/`.
- تعديل لوحة الإدارة: `admin-web/src/app/dashboard/...`.
- تعديل بيانات الإدارة المسموحة: `admin-web/src/app/api/data/route.ts`.
- تعديل سكيمة Replit PostgreSQL: `server/schema.sql` ثم `npm run db:push`.
- تعديل push notifications: `services/pushNotificationsService.ts`, `services/pushTokensService.ts`, `admin-web/src/app/api/send-order-push/route.ts`.
